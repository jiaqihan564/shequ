前端依赖安装说明
==================

由于echarts 6.0.0与echarts-gl 2.0.9存在版本冲突，需要使用以下命令安装：

方法1（推荐）：使用 --legacy-peer-deps 标志
npm install --legacy-peer-deps

方法2：使用 --force 强制安装
npm install --force

方法3：降级echarts到5.x版本（如果需要）
1. 修改 package.json 中的 echarts 版本为 "^5.6.0"
2. 运行 npm install

推荐使用方法1，这样可以避免破坏依赖关系。

安装完成后运行：
npm run dev
